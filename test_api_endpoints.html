后端接口路径测试

1. 获取网点列表

路径: GET http://localhost:8000/point_list

说明: 获取所有网点列表

2. 获取网点详情

路径: POST http://localhost:8000/admin/getPointInfo

说明: 根据ID获取网点详细信息

3. 新增网点

路径: POST http://localhost:8000/admin/addPoint

说明: 新增网点信息

4. 修改网点

路径: POST http://localhost:8000/admin/updatePoint

说明: 修改网点信息
